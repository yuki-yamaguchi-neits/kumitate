# Kumitate

Kumitate is a lightweight WordPress production base for managing AI-generated HTML, CSS, and JavaScript as page sections.

## Structure

wp-content/
  themes/
    kumitate-blank/
  plugins/
    kumitate-sections/

## Packages

### Kumitate Blank

A super lightweight blank WordPress theme for Kumitate section-based pages.

### Kumitate Sections

A WordPress plugin for managing section-based HTML / CSS / JS inside fixed pages.

## Current status

Prototype / v0.1.0

## Initial scope

- Section-based HTML / CSS / JS management for WordPress pages
- Page-level JSON-LD
- Common parts
- Official log support
- Page data export

## Out of scope

- AI API integration
- SNS automation
- ACF / CPT integration
- Visual builder replacement
- General-purpose CMS

## Notes

This repository is currently for internal and small-team use.
