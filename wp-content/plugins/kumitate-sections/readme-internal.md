# Kumitate Sections

This plugin is part of Kumitate.

Initial scope:
- Section-based HTML/CSS/JS management for WordPress pages
- Page-level JSON-LD
- Common parts
- Official log support

Out of scope:
- ACF
- CPT
- AI API integration
- SNS automation
- Elementor replacement

## バージョン互換性について

Kumitateは、テーマ「Kumitate Blank」とプラグイン「Kumitate Sections」の組み合わせで動作します。それぞれ独立したバージョンを持ちますが、以下のセットでの利用を推奨します。

### 推奨セット (v0.1.2)
- **Kumitate Blank**: v0.1.1
- **Kumitate Sections**: v0.1.2

通常、管理画面のUI改善やAI連携機能のアップデートはプラグイン側（Kumitate Sections）のみで行われるため、テーマとプラグインのバージョンが一致しない場合があります。

将来的に特定のテーマバージョンが必須となる場合は、ここに「Kumitate Sections vX.X.X requires Kumitate Blank vY.Y.Y or later」のように明記します。
